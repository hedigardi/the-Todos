## Installation
1. Clone the Todos App repository:
   ```bash
   git clone https://github.com/hedigardi/the-Todos.git
    ```
2. Install the dependencies:
   ```bash
   npm install
    ```
3. Start the application:
   ```bash
   npm run start
    ```
